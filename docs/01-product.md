# Product Definition

## Vision
A privacy-first, self-hosted journaling platform with offline-first guarantees, rich media support, and seamless cross-device synchronization.

## Goals
- Offline-first journaling experience
- Cross-platform support (Web, Desktop, Mobile)
- Full user data ownership (self-hosted)
- Fast and reliable search
- Secure data handling

## Non-Goals (MVP)
- Real-time collaboration
- Social sharing features
- Multi-user shared workspaces

## Core Features (MVP)
- Create/edit journal entries
- Rich text editor (TipTap)
- Tags and metadata (mood, timestamps)
- Local-first storage
- Background sync
- Basic search

## Phase 2 Features
- End-to-end encryption (E2EE)
- Attachments (images/audio)
- AI-assisted journaling (local models)
- Export (Markdown, JSON, PDF)
- Version history

## Target Users
- Privacy-conscious individuals
- Developers / self-hosters
- Personal knowledge management users
